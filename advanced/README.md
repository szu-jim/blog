个人博客
===============================

用于个人学习

使用 `Yii 2` 框架开发

目录
-------------------

```
common
    config/              配置
    models/              建立数据表的数据模型，取出数据供前后台使用  
console
    config/              配置
    controllers/         主机控制器
    migrations/          数据库迁移
    models/              特定模型类
    runtime/             运行时产生的文件
backend
    assets/              静态文件的配置
    config/              配置
    controllers/         控制器，控制数据存取，渲染页面
    models/              建立表单模型，编写功能实现所需的逻辑
    runtime/             运行时产生的文件    
    views/               视图
    web/                 存放静态资源
frontend
    assets/              静态文件的配置
    config/              配置
    controllers/         控制器，控制数据存取，渲染页面
    models/              建立表单模型，编写功能实现所需的逻辑
    runtime/             运行时产生的文件    
    views/               视图
    web/                 存放静态资源
    widgets/             需使用的组件
vendor/                  依赖的第三方包
environments/            覆盖基础环境
```
